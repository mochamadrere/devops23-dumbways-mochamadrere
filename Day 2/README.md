### Penjelasan :
- 192.168.11.0/30 bagaimana cara menghitung subnet mask nya?


  11111111.11111111.11111111.**11111100**  => 128+64+32+16+8+4 = **252**

  Jadi, karena kita menggunakan IP Kelas C, Subnet mask dari /30 adalah **255.255.255.252**
